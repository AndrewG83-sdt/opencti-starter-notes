# OpenCTI Starter Notes

Repository di appunti iniziali e contesto tecnico legati a un percorso di apprendimento su OpenCTI.

## Obiettivo

Questo repository serve a mostrare interesse e familiarita crescente con strumenti e workflow CTI, anche quando il materiale e ancora in forma di note, appunti o base di partenza.

## Come va letto

- come repository di studio tecnico
- come appoggio a futuri workflow piu strutturati
- come parte dell'area tooling e platform-oriented del mio profilo

## Perche ha valore nel profilo

Non tutti i repository devono essere prodotti completi. Alcuni aiutano a mostrare il percorso verso stack e strumenti piu evoluti, in questo caso nell'area cyber threat intelligence.

## Contesto

Materiale di studio, laboratorio e apprendimento. Il repository e mantenuto pubblico per documentare il percorso tecnico.
